URL ที่ได้จากการ Deploy Google Apps Script เป็น Web App

  1. เปิด Google Sheets และสร้าง Spreadsheet ใหม่ — จดจำ Sheet ID (จาก URL)
  2. ไปที่ Google Apps Script → สร้างโปรเจ็กต์ใหม่
  3. วางโค้ดจากไฟล์ Code.gs ที่ดาวน์โหลด
  4. แก้ไข SHEET_ID ให้เป็น ID ของ Google Sheet คุณ
  5. รัน setupSheets() เพื่อสร้าง Sheet header และข้อมูลตัวอย่าง
  6. Deploy → New Deployment → Web App → Execute as "Me" → Access "Anyone"
  7. คัดลอก URL ที่ได้มาใส่ในช่องด้านบน
