⚙️ วิธีตั้งค่า Google Apps Script

✅ ทำครั้งเดียว ใช้ได้ตลอด — ข้อมูลทั้งหมดจะเก็บใน Google Sheets ของคุณเอง
  1. เปิด Google Sheets ใหม่ แล้วตั้งชื่อ เช่น "ระบบพัสดุโรงเรียน"
  2. คลิกเมนู Extensions → Apps Script
  3. ลบโค้ดเดิมทั้งหมด แล้ว วางโค้ดจากไฟล์ apps_script.gs ที่ได้รับ
  4. คลิก 💾 Save แล้วคลิก ▶ Run → initSheets เพื่อสร้าง Sheet ครั้งแรก
    อนุมัติสิทธิ์เข้าถึง Google Sheets เมื่อระบบถาม
  5. คลิก Deploy → New deployment
    → เลือก Type: Web app
    → Execute as: Me
    → Who has access: Anyone (หรือ Anyone with Google account)
    → คลิก Deploy
  6. คัดลอก Web app URL ที่ได้ (ขึ้นต้นด้วย https://script.google.com/...)
  7. วาง URL นั้นในช่องด้านบนสุดของหน้านี้ แล้วคลิก ✓ บันทึก
💡 หมายเหตุ: ทุกครั้งที่แก้ไข Apps Script ต้อง Deploy ใหม่ (New deployment) เพื่อให้การเปลี่ยนแปลงมีผล
